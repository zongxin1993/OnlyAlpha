# Market Profile Capabilities

Capability 是通过行为场景证明的能力，不是愿望清单。Stable Profile 的每个 true 字段必须在绑定 Pack 中存在非 unsupported Scenario；`OnlyMarketConformanceRegistry.validate_capabilities` 执行该门禁。

当前内建能力用于描述已存在的规则模型，但所有版本仍为 Experimental，因为正式 Engine Conformance Packs 尚未完成。Futures 的 close-today/funding/liquidation、Crypto maker-taker/funding、多币种换算均为 false。
